Ключи и параметры запуска программы:

"Название_участка"          - обязательный параметр. Запуск программы для загрузки станций участка,
                              данные о котором берутся из файла, путь к которому указан в реестре: 
                              [HKEY_LOCAL_MACHINE\SOFTWARE\NEMAN\INI]
                              "Название_участка"="C:\\Неман\\...\\Ini" - путь к файлу VIS_Название_участка.cnf

/user:NEMAN /password:NEMAN - запуск прграммы с именем пользователя и пароля для разрешения отправки 
                              команд ТУ. Команды по станциям берутся из баз данный, путь к которым 
                              указан в реестре:
                              [HKEY_LOCAL_MACHINE\SOFTWARE\NEMAN\COMMANDS]
                              "Название_участка"="c:\\Неман\\...\\Commands"

/testport:5555             - запуск программы с открытием порта для подключения клиента автотеста

Для работы с SectionModel, запущенном на машине в сети, прописывается путь в реестре: 
[HKEY_LOCAL_MACHINE\SOFTWARE\NEMAN\SMSERVERS]
"Название_участка" = "ip-адрес" (или "имя_машины")
ip-адрес - адрес машины, с запущенным SectionModel участка.


Параметры в файле .cnf:

[Section]
Name=ДНЦ Житковичи-Муляр    - отображаемое имя АРМа в программе
StationCount=5              - кол-во загружаемых станций   
ImageCount=2                - кол-во загружаемых видов
Server=10.20.47.110:5055    - адрес:порт подключения к SectionModel (по умолч. 127.0.0.1:5050)

SourceLocationPath=C:\Неман\БелГУТ-ЦУП\Graphics\Житковичи-Муляр                 - путь к размещяемым папкам с графикой станций
ImagesLocationPath=C:\Неман\БелГУТ-ЦУП\Graphics\Images                          - имиджи станций
ColourSetsLocationPath=C:\Неман\БелГУТ-ЦУП\Graphics\Colours                     - цветовые наборы
CommandDatabaseLocationPath=c:\Неман\БелГУТ-ЦУП\Commands\Житковичи-Муляр        - путь к базам данных команд
LoadingErrorsLogPath=c:\Неман\БелГУТ-ЦУП\VIS_Log\Житковичи-Муляр\Loading        - путь к логу загрузки программы
UseCodes=1                  - использовать коды станций вместо имен

[StationCodes]
0=130206                    - код станции

[StationNames]
0=Брест-ВосточныйК          - название станции

[ImagePrefixes]
0=sg                        - префикс имени файла вида

[ImageScales]
0=x:1,00  y:1,00            - масштабирование по осям Х-У вида (по умолч. х=1, у=1)

[ImageNames]
0=крупный                   - название вида

[UIParams]
OldMapHeight=75              - высота MAP
IsMapVisible=0               - отображается ли MAP (1/0)
ImageIndex=0                 - последний вид
StationIndex=0               - последняя станция
IsMaximised=0                - окно распахнуто (1/0)
Left=2056                    - позиция окна
Top=16                       - // - // -
Width=1036                   - размеры окна
Height=549                   - // - // -
ROW_HEIGHT_CAPTION=30        - параметры меню команд
ROW_HEIGHT_COMMON=22         - // - // -
ROW_CAPTION_FSIZE=18         - // - // -
ROW_COMMON_FSIZE=14          - // - // -
ReadPeriod=50                - период опроса SectionModel, мс. (по умолч. 200)
DrawPeriod=50                - период прорисовки картинки, мс. (по умолч. 100)
FastMode=0                   - режим быстрой отрисовки, по изменению состояния, а не под фазы мигания (1/0, по умолч. 0)
